# 🟠 Don't Burn Crypto

## Before you burn it. Donate it.

Every year, millions of dollars in cryptocurrency are intentionally burned.

This project proposes a simple alternative.

Instead of destroying value forever, donate it.

## Website

https://dontburncrypto.org

## Mission

Help transform burned cryptocurrency into something that creates value.

## How it works

- Choose your preferred blockchain.
- Copy the donation address or scan the QR code.
- Send any amount.
- Every donation is recorded permanently on the blockchain.

## Transparency

All wallet addresses are public.

Every transaction can be independently verified on its respective blockchain explorer.

## Open Source

The website source code is public to encourage transparency and trust.
